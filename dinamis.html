Web Dinamis

Project information

  • Category: Dinamic Web
  • Project date: 16 December, 2020

A dynamic web that contains a story blog where there are three actors, namely readers, writers, admins. Authors must be logged in to be able to post stories. Where in this task I made for the author's side. Readers can read, search, comment on posts made by the author. Admin manages the continuity of the web such as adding an author account.
